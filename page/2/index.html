Hexo + Next个人博客的搭建

发表于 2017-01-09 | | 阅读次数

Hexo安装

前期准备

1、注册github账号
2、建立远程仓库(仓库名：flymengfei.github.io)
3、下载和安装Git

创建本地仓库

1、创建文件夹hexo(位置自由选择)，在\hexo文件夹内，点击鼠标右键，选择Git Bash，执行以下命令：
$git init
$ls -ah
$ssh-keygen -t rsa -C "fly.mengfei@gmail.com"
2、在C:\Users\mengfei目录下生成一个.ssh的文件夹，进入.ssh文件夹，使用Notepad++打开id_rsa.pub文件，复制全部内容(key)。登陆github，点击自己头像下面的Settings，选择SSH and GPG keys，点击有上方的New SSH key，Title随便填，Key：复制上面的key(id_rsa.pub文件里的全部内容)。之后在fly.mengfei@gmail.com会收到一封验证邮件，验证即可。为了验证是否成功，在git bash下输入命令：
$ssh -T git@github.com
显示You’ve successfully flymengfei, but GitHub does not provide shell access
这就表示已成功连上github

hexo配置

1、下载node.js
2、安装，打开\hexo文件夹，鼠标右键选择Git Bash，执行以下命令：
$npm install hexo-cli -g
$hexo init
$npm install
$hexo g
$hexo s
然后用浏览器访问http://localhost:4000/
3、hexo命令
$hexo new "postName" #新建文章（hexo\source_posts\postName.md）
$hexo new page "pageName" #新建页面（hexo\source\pageName）
$hexo g #生成静态页面至public目录
$hexo s #开启预览访问端口（默认端口4000）
$hexo d #将.deploy目录部署到GitHub

上传本地仓库

1、设置username和email，执行以下命令：
‘$git config –global user.name “flymengfei”‘
‘$git config –global user.email “fly.mengfei@gmail.com”‘
2、编辑_config.yml文件：编辑\hexo下的_config.yml文件，在文件的最下方，添加如下配置：
deploy:
type: git
repository: git@github.com:flymengfei/flymengfei.github.io.git
branch: master
3、上传：
$npm install hexo-deployer-git --save
$hexo d -g
4、刷新博客：
hexo clean
hexo d -g
