安装教程

安装Anaconda，自带Jupyter

更改Jupyter工作空间

  • 在cmd中输入jupyter notebook --generate-config，找到配置文件ipython_notebook_config.py存在的位置
  • 在其配置文件ipython_notebook_config.py中找到下面两段，加粗部分为更改部分，注意去掉底下一行的注释#（我的在194-195行）
    The directory to use for notebooks and kernels.
    c.NotebookApp.notebook_dir = 'D:\Jupyter'

为Jupyter添加目录功能

  • 利用Anaconda安装Jupyter Notebook extensions
    conda install -c conda-forge jupyter_contrib_nbextensions
  • 打开Jupyter Notebook，在它的（新增的）Nbextensions标签下勾选“Table of Contents(2)”

Python 3 is no longer available as a kernel in Jupyter Notebook after updating Anaconda

conda remove nb_conda_kernels
conda create -n jupyter notebook
